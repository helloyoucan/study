#### web与native通讯

1. web 发送到 navite

```javascript
// 打电话给老师
{
    type:'phone_to_teacher',
    status:1
}
// 播放单词视频
{
    type:'video',
    status:3,
    url:''
}
// 切换布局
{
    type:'status',
    status:1
}
{ // 课堂结束，回到课程列表
    type:'end'
}
```

2. navite发送到web

```javascript
{ // 下一个环节（老师出场后，进行单词学习）
    type:'next'
}
```
